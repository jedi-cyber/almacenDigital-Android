# Mejora de estilo profesional

## Objetivo

Elevar la apariencia de la app movil para que se sienta como una herramienta operativa profesional, no como una pantalla simple de pruebas.

## Cambios aplicados

- Paleta mas sobria:
  - fondo gris claro
  - superficies blancas
  - primario verde industrial
  - acento cobre discreto
- Cabecera principal con bloque visual fuerte.
- Tarjetas con bordes, elevacion y mejor espaciado.
- Botones con altura tactil, texto sin mayusculas forzadas y jerarquia primaria/secundaria.
- Campos con fondo propio y mejor padding.
- Estados de informacion/error con fondos diferenciados.
- Textos del dashboard reorientados al objetivo del proyecto:
  - buscar productos
  - consultar ubicacion
  - seguir ruta guiada
  - usar ruta guiada movil y Ruta 3D sin controles administrativos

## Criterio visual

La app debe parecer una herramienta de almacen:

- clara
- tactil
- legible
- sobria
- rapida de usar en campo

No debe parecer una copia web ni una maqueta simple.
